The A-maze-ing Race | Maze-solving Robot

The "The A-maze-ing Race" project involved a maze which required us to create a robot which can solve through the maze in the shortest amount of time.

Along the maze, various pieces of coloured paper instruct our robot react accordingly in order to complete the race. The colours help determine which direction the robot has to move in through the maze at a particular way point.


We had to build custom sensors using LEDs, LDRs, IR Emitter/Detector, then integrate them with the pre-built sensors in order to:

  • Not bump into walls
  • Stop when a line is detected
  • Turn based on the colour beneath the robot
  • Play a victory tune at the end of the maze

Obstacle Course Robot

The objective of this project is to create a robot that is powered by a 9.0V battery and must navigate through an obstacle course with a slope and bump. The robot must then stop 5 cm away from a wall and deliver a ping pong ball over it.

The robot must be smaller than 30cm x 30cm x 30cm, and can only be constructed using the provided materials.

The time limit for completing the course is 60 seconds.
